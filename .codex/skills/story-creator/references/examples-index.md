# Canonical Examples Index

## Как пользоваться индексом

Для каждой задачи определи: тип (аватар / навык / сценарий / prompt) и масштаб (лёгкий / средний / тяжёлый). Затем найди 1-3 подходящих эталона из списка ниже.
