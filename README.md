# AI Dungeon for ALL AI (Simulator Life)

Добро пожаловать в проект симуляции "Simulator Life"! Здесь собраны промпты, аватары, сценарии и системные настройки для создания продвинутых ролевых текстовых игр на базе различных AI.

## Состав проекта
- **Avatars/** - Большая коллекция аватаров и детальных профилей персонажей для различных вселенных (Demon Slayer, Resident Evil, Death Note, Isekai, Konosuba, etc.).
- **System/** - Системные промпты (разные версии `prompt_simulator` и `prompt_aigm`) для настройки игрового мастера.
- **Skills/** - Описания специальных навыков и способностей для персонажей.
- Текстовые файлы симуляций и предысторий для множества персонажей.

## Как использовать
1. Скопируйте нужный системный промпт (например, `System/prompt_simulator_v8.json`).
2. Вставьте его в ваш любимый AI (ChatGPT, Claude, Gemini и т.д.) в виде системной инструкции или первого промпта.
3. Выберите или создайте аватар персонажа и отправьте его AI для начала игры.
4. Наслаждайтесь бесконечными и глубокими приключениями!
